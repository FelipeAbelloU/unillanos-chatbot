================================================================

Abre una terminal en VS Code (Ctrl + ñ) y ejecuta en orden:

  venv\Scripts\activate  →  python ui\django_chatbot\run.py  →  abrir 127.0.0.1:8000


================================================================
2. CAMBIAR VERSION DEL MODELO
================================================================

Editar el archivo config\config.yaml y cambiar la linea:
  checkpoint_path: "data/checkpoints/unillanos-v1"

Por la version que quieras usar:
  checkpoint_path: "data/checkpoints/unillanos-v2"

Guardar el archivo y reiniciar el servidor.

Versiones disponibles en data/checkpoints/:
  - unillanos-v1 : primer entrenamiento (20 PDFs, 228 pares QA)
  - unillanos-v2 : segundo entrenamiento (34 PDFs, 384 pares QA, mejoras)

4. PROBLEMAS FRECUENTES Y SOLUCIONES
================================================================

PROBLEMA: ModuleNotFoundError: No module named 'django'
SOLUCION: Olvidaste activar el venv.
          Ejecuta primero: venv\Scripts\activate

PROBLEMA: "El modelo aun no esta disponible"
SOLUCION: Falta configurar el checkpoint.
          Edita config\config.yaml y pon la ruta correcta en checkpoint_path.

PROBLEMA: La pagina no carga en el navegador
SOLUCION: El servidor no esta corriendo. Revisa la terminal.
          Si hay error, copialo y consulta a Claude.

PROBLEMA: Puerto 8000 ocupado
SOLUCION: Usa otro puerto:
          python ui\django_chatbot\run.py --port 8080
          Luego abre: 127.0.0.1:8080

PROBLEMA: El modelo tarda mucho en cargar (primera vez)
SOLUCION: Normal. El modelo pesa ~3 GB y tarda 1-3 minutos en cargar a RAM.
          Las siguientes preguntas responden mas rapido.

6. SCRIPT DESCARGA CHECKPOINT DESDE COLAB (UNO POR UNO)
================================================================

Cuando el entrenamiento en Colab termina, pega esto en una celda nueva:

--------------------------------------------------------------
from google.colab import files
from pathlib import Path

OUTPUT_DIR = "/content/unillanos-v2"

archivos = sorted([
    f for f in Path(OUTPUT_DIR).iterdir()
    if f.is_file() and not f.name.startswith("_")
])

print(f"{len(archivos)} archivos para descargar.")
print("El navegador abrira un dialogo por cada archivo.")
print("Acepta cada descarga ANTES de presionar Enter.\n")

for i, f in enumerate(archivos, 1):
    size_mb = f.stat().st_size / 1e6
    print(f"[{i}/{len(archivos)}] Descargando: {f.name}  ({size_mb:.0f} MB)")
    files.download(str(f))
    if size_mb > 50:
        input("    Espera a que la barra de descarga llegue al 100% y presiona Enter...")
    else:
        print("    (archivo pequeno, continua automaticamente)")

print("\nTodos los archivos descargados.")
print("Colocalos en: CANUTO\\data\\checkpoints\\unillanos-v2\\")
--------------------------------------------------------------

Despues de descargar, mueve todos los archivos a:
  C:\Users\Equipo\OneDrive\Documentos\Tesis\CANUTO\data\checkpoints\unillanos-v2\

Y actualiza config\config.yaml:
